## 🚀 What is Altude?

Altude unlocks the blockchain. We provide drop-in SDKs that let you build Web3 products without worrying about gas, wallets, or infrastructure.  All while maintaining custody and security of your customers.

Whether you're building a game, an app, or a platform — Altude gives you a complete toolkit to deliver seamless, real-time blockchain features with just a few lines of code.

- [x] Gasless transactions (Create Account, Send) 
- [ ] NFT creation and minting (including cNFTs)  
- [ ] Smart accounts (EOA)

Easy for builders and even easier for users
